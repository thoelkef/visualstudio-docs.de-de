# <a name="c26456-donthideoperators"></a>C26456 DONT_HIDE_OPERATORS

Ausblenden von Basismethoden, die nicht von virtuellen ist fehleranfällig und wird Code schwieriger zu lesen.

[C++-Core-Richtlinie für diese Warnung](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#c128-virtual-functions-should-specify-exactly-one-of-virtual-override-or-final)
