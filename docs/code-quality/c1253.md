# <a name="c1253"></a>C1253
Warnung C1253: Die Modelldatei kann nicht geladen werden.

 Das Code Analysetool meldet diese Warnung, wenn ein interner Fehler in der Modelldatei vorliegt, nicht im analysierten Code.
