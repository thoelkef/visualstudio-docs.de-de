# <a name="c1255"></a>C1255
Warnung C1255: PCH-Daten für Plug-in hat die falsche Länge.  
  
 Der Code Analysis-Tool meldet diese Warnung aus, wenn es ist ein interner Fehler im Tool, nicht in den Code analysiert wird.
