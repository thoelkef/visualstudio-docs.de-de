# <a name="c26483-staticindexoutofrange"></a>C26483 STATIC_INDEX_OUT_OF_RANGE

Wert der Wert %value% ist außerhalb des gültigen Bereichs (0, gebundenen %) der Variablen "% Variable%". Index nur mithilfe konstanter Ausdrücke, die innerhalb der Grenzen des Arrays sind Arrays. Finden Sie unter [Bounds.2 für C++-Core-Richtlinien](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#SS-bounds)
