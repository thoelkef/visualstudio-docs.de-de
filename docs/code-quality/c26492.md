# <a name="c26492-no_const_cast"></a>C26492 NO_CONST_CAST

Verwenden Sie nicht `const_cast` zum Umwandeln von `const`. Weitere Informationen finden Sie in den [C++ Core-Richtlinien Type.3](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#SS-type).
