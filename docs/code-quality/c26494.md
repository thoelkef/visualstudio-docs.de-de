# <a name="c26494-varusebeforeinit"></a>C26494 VAR_USE_BEFORE_INIT

Die Variable "% Variable%" ist nicht initialisiert. Ein Objekt immer initialisiert werden. Finden Sie unter [C++ Core Richtlinien Type.5](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#SS-type).
