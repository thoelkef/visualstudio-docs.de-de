# <a name="c26496-useconstforvariable"></a>C26496 USE_CONST_FOR_VARIABLE
  Die Variable "% Variable%" wird nur einmal zugewiesen, markieren Sie sie als `const`. Finden Sie unter [C++ Core Richtlinien con.4](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#con4-use-const-to-define-objects-with-values-that-do-not-change-after-construction).
