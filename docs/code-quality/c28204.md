# C28204
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28204: \<Funktion\> : Nur eine dieser Überschreibung und diejenige bei \<filename\>\(\<line\>\) sind für \<paramname\> mit einer Anmerkung versehen: beide müssen oder keine darf mit einer Anmerkung versehen werden.  
  
 Diese Warnung wird ausgegeben, wenn ein Fehler in den Anmerkungen gibt.
