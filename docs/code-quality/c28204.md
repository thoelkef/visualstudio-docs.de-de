# <a name="c28204"></a>C28204
Warnung C28204: \<function >: nur eine dieser über Ladungen und die bei \<filename > (\<line >) werden \<paramname > mit Anmerkungen versehen: beide oder keine von beiden müssen mit Anmerkungen versehen werden.

 Diese Warnung wird gemeldet, wenn ein Fehler in den Anmerkungen vorliegt.
