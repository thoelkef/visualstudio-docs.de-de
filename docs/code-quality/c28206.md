# <a name="c28206"></a>C28206
Warnung C28206: \<expression >: der linke Operand zeigt auf eine Struktur, verwenden Sie `->`.

 Diese Warnung wird gemeldet, wenn die Schreibweise der Struktur Zeiger dereferenzierungsnotation `->` erwartet wurde.
