# <a name="c28207"></a>C28207
Warnung C28207: \<Ausdruck >: Linker Operand eine Struktur ist, verwenden.

 Diese Warnung wird ausgegeben, wenn eine Struktur zu dereferenzieren Punkt (.) wurde erwartet.
