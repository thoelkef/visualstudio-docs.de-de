# <a name="c28216"></a>C28216
Warnung C28216: Die \_check @ no__t-1Return @ no__t-2-Anmerkung gilt nur für nach Bedingungen für den bestimmten Funktionsparameter.

 Die `_Check_return_`-Anmerkung wurde in einem anderen Kontext als `__post` angewendet. möglicherweise wird ein `__post`-Modifizierer (oder `__drv_out`) benötigt, oder er kann falsch platziert werden.
