# <a name="c28217"></a>C28217
Warnung C28217: für die Funktion, die Anzahl der Parameter für die Anmerkung stimmt nicht überein, die in der Datei gefundenen  
  
 Die Anmerkungen für die aktuelle Zeile und in der Zeile in der Nachricht sind inkonsistent. Dies dürfte nicht möglich, wenn die standard-Makros zur Anmerkungen verwendet werden; Diese Warnung wird nicht erwartet, während normaler Verwendung betrachtet werden.
