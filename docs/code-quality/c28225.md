# <a name="c28225"></a>C28225
Warnung C28225: In der Anmerkung wurde nicht die richtige Anzahl an Parametern gefunden  
  
 Diese Warnung gibt an, dass die benannten Anmerkung mit der falschen Anzahl von Parametern verwendet wird. Dies dürfte nicht möglich, wenn die standard-Makros zur Anmerkungen verwendet werden; Diese Warnung wird nicht erwartet, in die typische Verwendung betrachtet werden.
