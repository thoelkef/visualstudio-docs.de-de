# <a name="c28230"></a>C28230
Warnung C28230: Der Parametertyp weist keinen Member auf.

 Diese Warnung gibt an, dass ein Argument an eine Anmerkung versucht, den Zugriff auf eine `struct`, `class`, oder `union` und das benannte Element ist nicht vorhanden.
