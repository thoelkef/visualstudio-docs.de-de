# C28246
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Warnung C28246: Die Anmerkung für Funktion '\<Name\>', Parameter '\<Parameter\>' stimmt nicht mit dem Typ des Parameters überein.  
  
 Ein \_\_deref Operator wurde auf einen Nichtzeigertyp angewendet, als eine Anmerkung zu erstellen.
