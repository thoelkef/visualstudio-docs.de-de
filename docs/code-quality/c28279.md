# <a name="c28279"></a>C28279
Warnung C28279: Für Symbol wurde ein 'begin' ohne zugehöriges 'end' gefunden.

 Die Anmerkung-Sprache unterstützt ein Begin-und End-Konstrukt (`{` und `}` in C), und die Kopplung ist unausgeglichen geworden. Diese Situation kann vermieden werden, wenn die Makros verwendet werden.
