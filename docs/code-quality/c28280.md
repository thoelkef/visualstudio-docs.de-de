# <a name="c28280"></a>C28280
Warnung C28280: Für das Symbol wurde ein 'end' ohne zugehöriges 'begin' gefunden.  
  
 Die Anmerkung-Sprache unterstützt eine Begin- und End (`{` und `}` in C#) erstellen und die Kopplung unausgeglichene geworden. Diese Situation kann vermieden werden, wenn die Makros verwendet werden.
