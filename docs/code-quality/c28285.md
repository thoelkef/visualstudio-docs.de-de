Warnung C28285: Syntaxfehler im Parameter der Funktion  
  
 Der Code Analysis-Tool meldet diese Warnung, wenn ein mögliche Fehler in der Modelldatei entdeckt wird. Einige Quellfehler Datei können auch dazu führen, dass solche Fehler.
