# <a name="c28289"></a>C28289
Warnung C28289: Funktion: ReadableTo oder WritableTo verfügte nicht über eine Limit-Spezifikation als Parameter  
  
 Der Code Analysis-Tool meldet diese Warnung aus, wenn die Anmerkung Funktionsparameter/miscoded ist, wie bereits erwähnt.
