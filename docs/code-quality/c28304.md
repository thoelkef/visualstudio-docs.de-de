# <a name="c28304"></a>C28304
Warnung C28304: Für C++ Reference-Parameter < parameter_name > wurde ein nicht ordnungsgemäß platzierter `_Notref_`-Operator gefunden, der auf \<token > angewendet wurde.

 Der `_Notref_`-Operator sollte nur in besonderen Fällen verwendet werden C++ , in denen Verweis Parameter und nur in vom System bereitgestellten Makros verwendet werden. Es muss direkt ein `_Deref_`-Operator oder eine funktionale Anmerkung befolgt werden.
