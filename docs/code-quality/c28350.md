# <a name="c28350"></a>C28350
Warnung C28350: Die Anmerkung \<Anmerkung > beschreibt eine Situation, die nicht bedingt anwendbar ist.

 In der Regel wird diese Warnung generiert, wenn eine Anmerkung angewendet wird, in denen die C-/C++-Typ überprüft wird.
