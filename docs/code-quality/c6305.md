# <a name="c6305"></a>C6305
Warnung C6305: Potenzieller Konflikt zwischen sizeof-Menge und countof-Menge.

 Diese Warnung gibt an, dass eine Variable mit einem `sizeof` Ergebnis wird hinzugef√ºgt oder von einem Zeiger subtrahiert oder `countof` Ausdruck. Dadurch werden unerwartete Skalierung in Zeigerarithmetik.

```

{
  int cb=sizeof(int);
```
