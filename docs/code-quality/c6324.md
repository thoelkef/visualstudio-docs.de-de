# <a name="c6324"></a>C6324
Warnung C6324: potenziell falsche Verwendung von \<Funktion1 >: wollten Sie verwenden \<Funktion2 >?  
  
 Diese Warnung gibt an, dass eine Zeichenfolge Copy-Funktion verwendet wurde, in denen eine Funktion zum Zeichenfolgenvergleich wurden sollten. Falsche Verwendung der Funktion kann dazu f√ºhren, dass einen unerwarteter logischer Fehler.  
  
## <a name="example"></a>Beispiel  
 Der folgende Code generiert diese Warnung:  
  
```  
#include <string.h>  
  
void f(char *title )  
{  
  if (strcpy (title, "Manager") == 0) // warning 6324  
  {  
    // code  
  }  
}  
```  
  
 Um die Warnung zu korrigieren, verwenden Sie `strcmp`, wie im folgenden Code dargestellt:  
  
```  
#include <string.h>  
  
void f(char *title )  
{  
  if (strcmp (title, "Manager") == 0)   
  {  
    // code  
  }  
}  
```
