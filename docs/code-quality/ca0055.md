# <a name="ca0055"></a>CA0055
Datei konnte nicht geladen:\<Pfad >

 Das Codeanalysetool konnte die Zieldatei, die angegeben wurde, für die Analyse zu laden. Dies tritt auf, wenn die Datei nicht gefunden wurde oder der Zugriff auf die Datei wurde verweigert. Das tritt in der Regel auf, wenn der Benutzer versucht, für die Eingabeassembly-Eigenschaft in Visual Studio zu überschreiben, oder fxcopcmd.exe manuell aufgerufen.
