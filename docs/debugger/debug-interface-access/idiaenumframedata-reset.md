# <a name="idiaenumframedatareset"></a>IDiaEnumFrameData::Reset
Setzt eine Enumerationsfolge auf den Anfang zurück.

## <a name="syntax"></a>Syntax

```
HRESULT Reset();
```

## <a name="return-value"></a>Rückgabewert
 Gibt `S_OK`zurück.
