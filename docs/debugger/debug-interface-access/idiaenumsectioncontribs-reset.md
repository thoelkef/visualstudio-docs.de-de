# IDiaEnumSectionContribs::Reset
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

Setzt die Enumerationsfolge auf den Anfang zurück.  
  
## Syntax  
  
```cpp#  
HRESULT Reset();  
```  
  
## Rückgabewert  
 Gibt `S_OK` zurück.
