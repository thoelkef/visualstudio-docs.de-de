# <a name="idiaenumsegmentsreset"></a>IDiaEnumSegments::Reset

## <a name="syntax"></a>Syntax

```C++
HRESULT Reset ();
```

## <a name="return-value"></a>Rückgabewert
 Gibt `S_OK`zurück.
