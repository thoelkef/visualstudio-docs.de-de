# <a name="edit-and-continue-dialog-box"></a>Dialogfeld „Bearbeiten und Fortsetzen“
(Nur nativ) Dieses Dialogfeld wird angezeigt, wenn Sie während des Debuggens Code bearbeitet haben und die Codeänderungen von der Funktion **Bearbeiten und Fortfahren** übernommen werden können. Weitere Informationen hierzu finden Sie unter [Bearbeiten und Fortfahren](../debugger/edit-and-continue.md).

 **Dieses Dialogfeld nie mehr anzeigen** Wenn Sie dieses Kontrollkästchen aktivieren, wendet " **Bearbeiten und Fortfahren** " Codeänderungen an, ohne in Zukunft eine Berechtigung zu erhalten. Sie können diese Warnmeldung erneut aktivieren, indem Sie im Dialogfeld **Optionen** im Ordner **Debuggen** auf der Seite **Bearbeiten und Fortfahren** die Optionen **Von Debugbefehlen aufgerufen** und **Zuerst fragen** auswählen.
