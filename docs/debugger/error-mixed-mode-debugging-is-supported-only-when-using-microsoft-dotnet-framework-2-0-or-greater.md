# <a name="error-mixed-mode-debugging-is-supported-only-when-using-microsoft-net-framework-20-or-greater"></a>Fehler: Debuggen im gemischten Modus wird nur bei Verwendung von Microsoft .NET Framework, Version 2.0 oder höher, unterstützt
Zum Debuggen von gemischtem System eigenem und verwaltetem Code benötigen Sie .NET Framework Version 2,0, 3,0. 3.5 oder 4 verwenden. Das Debuggen im gemischten Modus mit früheren Versionen des .NET Framework wird nicht unterstützt.

### <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler

- Aktualisieren Sie die .NET Framework auf Version 2,0, 3,0, 3,5 oder 4,0.
