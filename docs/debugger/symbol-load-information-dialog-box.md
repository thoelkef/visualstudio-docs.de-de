# <a name="symbol-load-information-dialog-box"></a>Dialogfeld "Symbolladeinformationen"
Das Dialogfeld **Symbolladeinformationen** wird angezeigt, wenn Sie mit der rechten Maustaste auf das Fenster **Module** klicken und die Option **Symbolladeinformationen** auswählen.

## <a name="uielement-list"></a>UIElement-Liste
 **Informationen** Zeigt die nach Symbol Dateien (PDB-Dateien) gesuchten Verzeichnisse auf der Grundlage von Visual Studio-Installationseinstellungen und Änderungen, die Sie an Symbol Einstellungen im Dialogfeld **Optionen** vorgenommen haben.

 **Symbol Einstellungen** Klicken Sie auf diese Schaltfläche, um Symbol Einstellungen im Dialogfeld **Optionen** zu ändern (Kategorie**Debuggen** , Seite **Symbole** ). Auf dieser Seite können Sie Symbolspeicherorte hinzufügen oder entfernen und, wenn Sie einen Symbolserver verwenden, einen lokalen Cachespeicherort angeben sowie steuern, wann die Symbolspeicherorte durchsucht werden.
