# A failure occurred writing to the resources file
Das Projektsystem konnte nicht in die RESOURCES\-Ausgabedatei schreiben.  Bei der Ressourcenvorbereitung wandelt das Projektsystem XML\-Dateien mit einer RESX\-Erweiterung für den .NET\-Ressourcen\-Manager in binäre Ressourcendateien um.  
  
 Eine für das Betriebssystem spezifische Ursache \(falls vorhanden\) wird an das Ende dieser Meldung angehängt.  Der Dateiname der RESX\-Datei wird ebenfalls angehängt.  
  
 Durch diesen Fehler schlägt der Buildprozess fehl.
