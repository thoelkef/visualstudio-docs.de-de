# CA0053
Die Regelassembly konnte nicht geladen werden.  
  
 Diese Warnung weist darauf hin, dass eine Regelassembly nicht durch Reflektion geladen werden konnte.  Dies kann passieren, wenn der Benutzer falsch erstellte benutzerdefinierte Regeln verwendet oder das Analysetool die Assembly aus anderen Gründen, z. B. wenn der Zugriff auf das Dateisystem verweigert wird, nicht laden kann.
