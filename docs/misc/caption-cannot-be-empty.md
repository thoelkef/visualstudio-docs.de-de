# Caption cannot be empty.
Im Allgemeinen tritt dieser Fehler auf, wenn für die Registerkarte der **Toolbox** kein Name eingegeben wurde.  
  
### So beheben Sie diesen Fehler  
  
1.  Geben Sie einen Namen für die Registerkarte der Toolbox ein.
