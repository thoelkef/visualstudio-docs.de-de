# Clipboard cannot be emptied.
Im Allgemeinen tritt dieser Fehler auf, wenn ein Element kopiert wurde, jedoch nicht in der Zwischenablage abgelegt werden kann, da diese bereits Informationen enthält, die nicht entfernt werden können.  
  
### So beheben Sie diesen Fehler  
  
1.  Schließen Sie andere Programme, die möglicherweise die Zwischenablage verwenden, und versuchen Sie es erneut.
