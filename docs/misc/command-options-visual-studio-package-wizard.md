# Befehlsoptionen, Assistent f&#252;r das Visual Studio-Paket
Mit diesem Schritt können Sie den Namen und die Befehls\-ID für einen Befehl definieren, der im Menü **Extras** von Visual Studio angezeigt wird.  
  
 **Befehlsname**  
 Text in diesem Feld wird wie folgt angezeigt:  
  
-   Der Name des Befehls.  
  
-   Die QuickInfo der Befehlsschaltfläche auf der Symbolleiste.  
  
 **Befehls\-ID**  
 -   Text in diesem Feld definiert die Befehls\-ID für den Befehl. Diese ID ist eine Konstante, die im generierten Code zum Verweisen auf dieses Menüelement verwendet wird.  
  
 **Abbrechen**  
 -   Bricht den Vorgang ab, und beendet den Assistenten.  
  
 **Vorheriges**  
 -   Wechselt zur vorherigen Seite des Assistenten.  
  
 **Fertig stellen**  
 -   Erstellt das VSPackage.  
  
 **Weiter**  
 -   Wechselt zur nächsten Seite des Assistenten.
