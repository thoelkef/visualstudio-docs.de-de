# Die bin&#228;re Lizenzdatei konnte nicht generiert werden. Die Ausgabedatei &quot;Datei&quot; konnte nicht ge&#246;ffnet werden. &lt;Ursache&gt;
Das Projektsystem konnte eine veraltete Kopie der Lizenzdatei nicht löschen. Bei der Lizenzvorbereitung wandelt das Projektsystem LICX\-Textdateien in binäre Lizenzdateien um, die vom .NET\-Lizenzierungssystem ausgewertet werden können.  
  
 **So beheben Sie diesen Fehler**  
  
-   Starten Sie [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] neu.  
  
     Der Buildprozess schlägt fehl, wenn dieser Fehler auftritt.
