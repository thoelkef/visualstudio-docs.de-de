# File is not a valid .NET Framework or COM component
Sie versuchen, eine Datei im Objektkatalog anzuzeigen, die kein gültiges Objekt enthält.  Im Objektkatalog können nur [!INCLUDE[dnprdnshort](../code-quality/includes/dnprdnshort_md.md)]\- und COM\-Komponenten angezeigt werden.  
  
### So beheben Sie diesen Fehler  
  
1.  Wenn Sie irrtümlich eine falsche Datei ausgewählt haben, klicken Sie auf die richtige Datei, um diese anzuzeigen.  
  
2.  Wenn Sie im Dialogfeld **Öffnen mit** versehentlich auf Objektkatalog geklickt haben, wählen Sie ein anderes Programm zum Öffnen der Datei aus.
