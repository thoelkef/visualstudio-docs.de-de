# File name or default path is too long.
Dieser Fehler tritt normalerweise auf, wenn versucht wurde, einen Webbericht über Codekommentare zu erstellen, der im Feld **Webseiten speichern in** des Dialogfelds **Erstellen von Kommentarwebseiten** angegebene Datei\- oder Pfadname jedoch zu lang ist.  
  
### So beheben Sie diesen Fehler  
  
1.  Geben Sie einen Datei\- oder Pfadnamen ein, der weniger als 260 Zeichen lang ist.
