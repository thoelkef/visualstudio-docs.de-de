# Invalid license data. Reinstall is required.
Dieser Fehler tritt im Allgemeinen auf, wenn die Lizenzschlüsseldaten in der Registrierung gelöscht oder beschädigt wurden.  
  
### So beheben Sie diesen Fehler  
  
1.  Installieren Sie das Produkt neu.
