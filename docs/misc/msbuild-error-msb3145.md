# MSBuild Error MSB3145
**MSB3145: Der Buildeingabeparameter "\<property\>\=\<value\>" ist keine Webadresse oder UNC\-Freigabe.**  
  
 Dieser Fehler tritt auf, wenn der Wert der Projekteigenschaft `SupportUrl`, `ComponentsUrl` oder `ApplicationUrl` ungültig ist.  Der Wert muss ein gültiger URI oder UNC\-Pfad sein.
