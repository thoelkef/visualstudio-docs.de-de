# MSBuild Error MSB3168
**MSB3168: Das doppelte \<Paket\>\-Element wird ignoriert.**  
  
 Diese Warnung wird ausgegeben, wenn Sie angegeben haben, dass der Bootstrapper zwei gleiche Bootstrapperpakete installieren soll.  Der Bootstrapper installiert in diesem Fall nur eine Instanz.  
  
### So beheben Sie diesen Fehler  
  
-   Weitere Informationen zur Ursache und zu möglichen Lösungen eines MSBuild\-Fehlers finden Sie unter [Additional Resources](../msbuild/additional-msbuild-resources.md).
