# New project platforms cannot be created because this new solution platform is not supported by any of the projects.
Dieser Fehler tritt auf, wenn Sie **Neue Projektplattformen erstellen** auswählen, die neue Projektmappenplattform jedoch in keinem Projekt unterstützt wird.  
  
### So beheben Sie diesen Fehler  
  
-   Deaktivieren Sie **Neue Projektplattformen erstellen**, und klicken Sie auf **OK**.  
  
     \- oder \-  
  
     Deaktivieren Sie **Neue Projektplattformen erstellen**, wählen Sie in der Dropdownliste den Namen einer anderen Plattform aus, und klicken Sie dann auf **OK**.
