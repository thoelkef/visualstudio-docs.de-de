# W&#228;hlen Sie mehr als eine Plattform aus. Wenn Sie nur an einer Plattform &#196;nderungen vornehmen m&#246;chten, klicken Sie im Dialogfeld „Mehrere Plattformen“ auf „Abbrechen“, und w&#228;hlen Sie die gew&#252;nschte Plattform in der Plattformen-Dropdownliste aus.
Dieser Fehler tritt im Allgemeinen auf, wenn im Dialogfeld **Mehrere Plattformen** eine einzelne Plattform ausgewählt wurde.  
  
### So beheben Sie diesen Fehler  
  
1.  Wählen Sie weitere Plattformen aus.  
  
     – oder –  
  
2.  Wenn Sie eine einzelne Plattform aktualisieren möchten, wählen Sie „Abbrechen“, und wählen Sie die richtige Plattform aus der Liste der Plattformen.
