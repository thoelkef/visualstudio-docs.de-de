# Please specify valid search string.
Im Allgemeinen tritt dieser Fehler auf, wenn eine Suche durchgeführt wird, ohne dass im Dialogfeld **Symbol suchen** in **Suchen nach** eine zu suchende Zeichenfolge eingegeben wurde.  
  
### So beheben Sie diesen Fehler  
  
1.  Geben Sie im Feld **Suchen nach** den richtigen Text ein, und wählen Sie erneut **Suchen** aus.
