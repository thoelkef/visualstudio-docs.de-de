# Projektbuildfehler PRJ0011
Möglicherweise sind nicht genügend Systemressourcen vorhanden. Ein Ereignis, das zum Starten eines Builds erforderlich ist, konnte nicht erstellt werden.  
  
 Wenn nicht genügend Systemressourcen verfügbar sind:  
  
-   Schließen Sie einige ausgeführte Anwendungen.  
  
-   Starten Sie Visual Studio neu, um Ressourcen freizugeben.  
  
-   Starten Sie den Computer neu.
