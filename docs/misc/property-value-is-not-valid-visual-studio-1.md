# Property value is not valid. (Visual Studio) (1)
Im Allgemeinen tritt dieser Fehler auf, wenn ein ungültiger Wert für eine Eigenschaft eingegeben wird.  Wenn es sich beispielsweise um eine boolesche Eigenschaft mit gültigen Werten von **True** und **False** handelt, ist der Wert **Vielleicht** für diese Eigenschaft ungültig.  
  
### So beheben Sie diesen Fehler  
  
1.  Geben Sie einen Wert ein, der für die Eigenschaft gültig ist.  
  
    > [!TIP]
    >  Suchen Sie in der Onlinedokumentation nach dem Namen der Eigenschaft, und ermitteln Sie die gültigen Werte.
