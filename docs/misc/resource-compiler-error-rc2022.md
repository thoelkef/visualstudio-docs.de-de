# Ressourcencompilerfehler RC2022
"Zahl": zu groß für ein Zeichen  
  
 Die Oktalzahl, die hinter einem umgekehrten Schrägstrich \(**\\**\) in einer Zeichen\- oder Zeichenfolgenkonstanten steht, war zu groß, um als ein Zeichen dargestellt werden zu können.
