# Ressourcencompiler: Fehler RC2145
Primäre Sprach\-ID ist zu groß.  
  
 Primäre Sprach\-ID muss eine gültige hexadezimale Sprach\-ID sein. Unter [Sprach\- und Länder\-\/Regionen\-Strings](/visual-cpp/c-runtime-library/locale-names-languages-and-country-region-strings) in der *Laufzeit\-Bibliothek\-Referenz* finden Sie eine Liste der gültigen Sprach\-IDs.
