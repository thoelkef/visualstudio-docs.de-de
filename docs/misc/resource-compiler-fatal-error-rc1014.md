# Ressourcencompiler: Schwerwiegender Fehler RC1014
Zu viele Includedateien: Tiefe \= 'Ebene'  
  
 Die Schachtelungstiefe von `#include`\-Direktiven ist zu umfassend.  
  
 Verwenden Sie geschachtelte Direktiven zum Einbeziehen geöffneter Dateien. Die Quelldatei, die die Direktive enthält, zählt als eine der Dateien.
