# Ressourcencompiler: Schwerwiegender Fehler RC1106
Ungültiger Schalter: Option  
  
 Die Option ist keine gültige Ressourcencompileroption.  
  
 Folgende Optionen sind gültig:  
  
 \/r  
  
 RES\-Datei ausgeben  
  
 \/v  
  
 \/Verbose \(Meldungen zum Druckfortschritt\)  
  
 \/d  
  
 Symbol definieren  
  
 \/fo  
  
 RES\-Datei umbenennen  
  
 \/l  
  
 Standardsprachkennung im Hexadezimalformat  
  
 \/I  
  
 Pfad für **INCLUDE**\-Suchen hinzufügen  
  
 \/x  
  
 **INCLUDE**\-Umgebungsvariable ignorieren  
  
 \/c  
  
 Von NLS Conversion zu verwendende Codepage definieren
