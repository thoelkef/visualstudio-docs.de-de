# Ressourcencompilerwarnung RC4006
\#undef erwartet einen Bezeichner  
  
 Der Name des Bezeichners, dessen Definition entfernt werden sollte, wurde nicht mit der `#undef`\-Anweisung angegeben. Die `#undef`\-Anweisung wurde ignoriert.
