# Ressourcencompiler: Warnung RC4205
Zeichenfolgenliteral länger als 256 Zeichen – Speichervorgang erfolgreich  
  
 Ein Zeichenfolgenliteral umfasst mehr als 256 Zeichen. Es wurde jedoch als Ganzes gespeichert.
