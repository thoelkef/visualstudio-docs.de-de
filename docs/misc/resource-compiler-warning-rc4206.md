# Ressourcencompiler: Warnung RC4206
Namenszeichenfolge zu lang – auf 256 Zeichen gekürzt  
  
 Eine Namenszeichenfolge hat den Grenzwert von 256 Zeichen überschritten. Sie wurde auf 256 Zeichen gekürzt.
