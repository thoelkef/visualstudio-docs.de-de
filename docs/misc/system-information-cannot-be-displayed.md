# System information cannot be displayed.
Informationen des Betriebssystems können nicht angezeigt werden. Möglicherweise fehlt ein Registrierungswert oder **msinfo32.exe** kann nicht ausgeführt werden.  
  
### So beheben Sie diesen Fehler  
  
1.  Installieren Sie Visual Studio neu.
