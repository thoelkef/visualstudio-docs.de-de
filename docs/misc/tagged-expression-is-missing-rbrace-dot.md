# Tagged expression is missing &#39;}&#39;.
Im Allgemeinen tritt dieser Fehler auf, wenn in einer Suchzeichenfolge eine öffnende geschweifte Klammer \(`{`\) angegeben wurde, die schließende geschweifte Klammer \(`}`\) jedoch fehlt.  
  
### So beheben Sie diesen Fehler  
  
1.  Wenn Sie das Literalzeichen `{` suchen möchten, verwenden Sie `\{`.  
  
2.  Wenn Sie einen Ausdruck markieren möchten, geben Sie die fehlende `}` ein.
