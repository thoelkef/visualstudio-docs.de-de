# The selected Add-In has not been confirmed to be &#39;Command Line Safe&#39;, and may require some user intervention (possible UI).
Dieser Fehler tritt normalerweise auf, wenn ein Add\-In aus der Befehlszeile \(DOS\-Eingabeaufforderung\) heraus verwendet wird, hierfür jedoch nicht geeignet ist.  Das Add\-In zeigt u. U. eine Benutzerschnittstelle an, die die Verwendung aus der Befehlszeile nicht unterstützt.  
  
### So beheben Sie diesen Fehler  
  
1.  Klicken Sie im Menü **Extras** auf **Add\-In\-Manager**.  
  
2.  Lassen Sie im Dialogfeld **Add\-In\-Manager** das Feld **Befehlszeile** für dieses Add\-In leer.
