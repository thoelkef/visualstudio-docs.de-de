# This program requires an application for class ID &lt;number&gt;.
Die gewünschte Anwendung wurde nicht gefunden.  Hierbei handelt es sich um einen Fehler auf Betriebssystem\-Ebene.  
  
### So beheben Sie diesen Fehler  
  
1.  Installieren Sie die betroffene Anwendung neu, und versuchen Sie es erneut.  
  
     \- oder \-  
  
     Installieren Sie Visual Studio neu, und führen Sie den Vorgang erneut aus.
