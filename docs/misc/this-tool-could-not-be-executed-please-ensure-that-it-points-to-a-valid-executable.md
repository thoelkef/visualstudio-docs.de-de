# This tool could not be executed. Please ensure that it points to a valid executable.
Im Allgemeinen tritt dieser Fehler auf, wenn Sie ein externes Tool aufrufen und dieses nicht ausgeführt werden kann.  Möglicherweise ist die angegebene Datei keine gültige ausführbare Datei oder Batchdatei.  
  
### So beheben Sie diesen Fehler  
  
1.  Stellen Sie sicher, dass das externe Tool eine gültige ausführbare Datei oder Batchdatei ist. Verwenden Sie dazu das Dialogfeld **Externe Tools**, auf das Sie im Menü **Extras** zugreifen.
