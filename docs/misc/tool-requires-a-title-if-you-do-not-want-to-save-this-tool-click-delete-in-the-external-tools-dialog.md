# Sie m&#252;ssen einen Namen f&#252;r dieses Tool angeben. Wenn Sie das Tool nicht speichern m&#246;chten, klicken Sie im Dialogfeld „Externe Tools“ auf „L&#246;schen“.
Dieser Fehler tritt normalerweise auf, wenn ein neues Tool zum Dialogfeld **Externe Tools** hinzugefügt, jedoch kein Titel angegeben wurde.  
  
### So beheben Sie diesen Fehler  
  
1.  Wenn Sie das Tool speichern möchten, klicken Sie auf **OK** und geben im Dialogfeld **Externe Tools** einen Titel für das Tool ein.  
  
2.  Wenn Sie das Tool nicht speichern möchten, klicken Sie auf **OK** und dann im Menü **Externe Tools** auf **Löschen**.
