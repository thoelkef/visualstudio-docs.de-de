# F&#252;r das Tool ist eine g&#252;ltige ausf&#252;hrbare Datei erforderlich. Wenn Sie das Tool nicht speichern m&#246;chten, klicken Sie im Dialogfeld „Externe Tools“ auf „L&#246;schen“.
Dieser Fehler tritt normalerweise auf, wenn ein Tool zum Dialogfeld **Externe Tools** hinzugefügt, jedoch keine ausführbare Datei angegeben wurde.  
  
### So beheben Sie diesen Fehler  
  
1.  Wenn Sie das Tool speichern möchten, klicken Sie auf **OK** und geben im Feld **Befehl** für das Tool einen Dateinamen ein, oder suchen Sie nach einer Datei, indem Sie auf die Schaltfläche mit den Auslassungszeichen klicken.  
  
2.  Wenn Sie das Tool nicht speichern möchten, klicken Sie auf **OK** und dann im Dialogfeld **Externe Tools** auf **Löschen**.
