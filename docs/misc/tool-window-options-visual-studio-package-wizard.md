# Optionen f&#252;r das Toolfenster, Assistent f&#252;r das Visual Studio-Paket
In diesem Schritt können Sie den Namen und die Befehls\-ID für das Toolfenster definieren.  
  
 **Fenstername**  
 Hier können Sie den Namen eingeben, der in der Titelleiste des Toolfensters angezeigt werden soll. Dieser Name gilt auch für den Menübefehl, mit dem das Toolfenster geöffnet wird.  
  
 Der Menübefehl für das Toolfenster wird im Menü **Ansicht** dem Untermenü **Weitere Fenster** hinzugefügt.  
  
 **Befehls\-ID**  
 Hier können Sie die Befehls\-ID für das Toolfenster definieren. Diese Konstante wird im generierten Code zum Verweisen auf dieses Menüelement verwendet.  
  
 **Abbrechen**  
 Bricht das Erstellen des VSPackage ab.  
  
 **Hinten**  
 Wechselt zur vorherigen Seite des Assistenten.  
  
 **Fertig stellen**  
 Erstellt das VSPackage.
