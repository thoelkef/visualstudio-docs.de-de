# Problembehandlung bei Ausnahmen: Microsoft.VisualStudio.Tools.Applications.Runtime.CannotFindStartupObjectAssemblyException
Diese Ausnahme wird ausgelöst, wenn die Hauptprojektmappenassembly nicht das im Anwendungsmanifest angegebene Startobjekt enthält.
