# Problembehandlung bei Ausnahmen: Microsoft.VisualStudio.Tools.Applications.Runtime.CannotLoadManifestException
Diese Ausnahme wird ausgelöst, wenn ein Manifest nicht geladen werden kann, weil es im Dokument nicht vorhanden ist bzw. beschädigt oder ungültig ist.
