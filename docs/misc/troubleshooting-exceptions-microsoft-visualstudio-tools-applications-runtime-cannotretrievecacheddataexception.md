# Problembehandlung bei Ausnahmen: Microsoft.VisualStudio.Tools.Applications.Runtime.CannotRetrieveCachedDataException
Diese Ausnahme wird ausgelöst, wenn die zwischengespeicherte Dateninsel in einem Dokument beschädigt ist oder fehlt.
