# Problembehandlung bei Ausnahmen: Microsoft.VisualStudio.Tools.Applications.Runtime.ControlNotFoundException
Eine <xref:Microsoft.VisualStudio.Tools.Applications.Runtime.ControlNotFoundException>\-Ausnahme wird ausgelöst, wenn im Code versucht wird, auf ein Steuerelement zuzugreifen, das nicht im Dokument ist. Das Steuerelement wurde womöglich vom Endbenutzer oder durch anderen Code entfernt. Es besteht keine Möglichkeit, vor dem Zugriff auf das Steuerelement dessen Anwesenheit sicherzustellen. Ohne das Steuerelement funktioniert die Projektmappe nicht.
