# Problembehandlung bei Ausnahmen: Microsoft.VisualStudio.Tools.Applications.Runtime.InvalidDeployManifestException
Diese Ausnahme wird ausgelöst, wenn das Anwendungsmanifest auf ein ungültiges Bereitstellungsmanifest verweist.
