# Problembehandlung bei Ausnahmen: Microsoft.VisualStudio.Tools.Applications.Runtime.NotAvailableOfflineException
Diese Ausnahme wird ausgelöst, wenn eine Assembly aufgerufen wird, die im Cache oder im Netzwerk nicht verfügbar ist.
