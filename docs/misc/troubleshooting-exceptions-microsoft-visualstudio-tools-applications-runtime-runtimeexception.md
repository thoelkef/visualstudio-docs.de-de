# Problembehandlung bei Ausnahmen: Microsoft.VisualStudio.Tools.Applications.Runtime.RuntimeException
Diese Ausnahme wird ausgelöst, wenn ein Laufzeitfehler auftritt, für den keine bestimmte Ursache festgestellt werden kann.
