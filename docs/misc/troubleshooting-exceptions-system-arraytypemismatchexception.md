# Problembehandlung bei Ausnahmen: System.ArrayTypeMismatchException
Eine <xref:System.ArrayTypeMismatchException>\-Ausnahme wird ausgelöst, wenn das System das Element nicht in den Typ konvertieren kann, der für das Array deklariert wurde.  
  
## Tipps  
 **Stellen Sie sicher, dass der Objekttyp in den Arraytyp konvertiert werden kann.**  
 Eine Konvertierung zwischen den zwei Typen muss möglich sein.  
  
 Visual Basic\-Benutzer finden weitere Informationen unter [Arrays](/dotnet/visual-basic/programming-guide/language-features/arrays/index).
