# Problembehandlung bei Ausnahmen: System.ComponentModel.Design.Serialization.CodeDomSerializerException
Diese Ausnahme wird ausgelöst, wenn das CodeDOM\-Serialisierungsprogramm auf einen Fehler trifft.  
  
## Hinweise  
 Diese Ausnahme ermöglicht es dem Serialisierungsprogramm, Informationen über Typ und Speicherort des Fehlers zurückzugeben.
