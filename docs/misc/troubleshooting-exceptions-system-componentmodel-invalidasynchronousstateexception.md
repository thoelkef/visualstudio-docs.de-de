# Problembehandlung bei Ausnahmen: System.ComponentModel.InvalidAsynchronousStateException
Diese Ausnahme wird ausgelöst, wenn der ausgeführte Thread nicht mehr gültig ist oder ein nicht behebbares Problem entwickelt hat.
