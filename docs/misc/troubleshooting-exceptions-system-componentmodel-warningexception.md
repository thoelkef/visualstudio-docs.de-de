# Problembehandlung bei Ausnahmen: System.ComponentModel.WarningException
Bezeichnet eine Ausnahme, die eher als Warnung denn als Fehler behandelt wird.  
  
## Hinweise  
 Wenn diese Ausnahme nicht abgefangen wird, wird sie dem Benutzer als Warnmeldung angezeigt.
