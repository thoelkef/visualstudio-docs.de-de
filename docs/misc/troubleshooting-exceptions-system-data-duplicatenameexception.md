# Problembehandlung bei Ausnahmen: System.Data.DuplicateNameException
Diese Ausnahme wird ausgelöst, wenn ein Datenbank\-Objektname während eines Vorgangs in einem mit <xref:System.Data.DataSet> verknüpften Objekt doppelt vorkommt und zum zweiten Mal erreicht wird.
