# Problembehandlung bei Ausnahmen: System.Data.SqlServerCe.SqlCeLockTimeoutException
Eine `System.Data.SqlServerCe.SqlCeLockTimeoutException`\-Ausnahme wird ausgelöst, wenn das Timeout für eine Sperre erreicht wurde und für SQL Server Compact beim Warten auf eine Sperre ein Timeout eingetreten ist. Der Standardwert für das Timeout beträgt 2000 Millisekunden. Weitere Informationen finden Sie in der Dokumentation zur [SQL Server Compact 3.5\-Klassenbibliothek](http://go.microsoft.com/fwlink/?LinkID=102595).
