# Problembehandlung bei Ausnahmen: System.Data.SqlServerCe.SqlCeTransactionInProgressException
Eine `System.Data.SqlServerCe.SqlCeTransactionInProgressException`\- Ausnahme tritt auf, wenn eine Anwendung versucht, während einer anderen Transaktion eine Datenbank zu ändern. Weitere Informationen finden Sie in der Dokumentation zur [SQL Server Compact 3.5\-Klassenbibliothek](http://go.microsoft.com/fwlink/?LinkID=102595).
