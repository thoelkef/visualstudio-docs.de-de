# Problembehandlung bei Ausnahmen: System.IdentityModel.Selectors.IdentityValidationException
Eine <xref:System.IdentityModel.Selectors.IdentityValidationException>\-Ausnahme wird ausgelöst, wenn das Empfängerzertifikat ungültig ist. Dies kann der Fall sein, wenn das Zertifikat nicht erfolgreich zum Stammzertifikat aufwärts verkettet.
