# Problembehandlung bei Ausnahmen: System.IdentityModel.Selectors.UntrustedRecipientException
Eine <xref:System.IdentityModel.Selectors.UntrustedRecipientException>\-Ausnahme wird ausgelöst, wenn ein Benutzer Informationen von einer Entität prüft, die ein Token anfordert, und dann entscheidet, der Entität nicht zu vertrauen. Der Benutzer klickt in der CardSpace\-Benutzeroberfläche auf **Windows CardSpace deaktivieren**.
