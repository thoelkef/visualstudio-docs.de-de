# Problembehandlung bei Ausnahmen: System.InvalidProgramException
Die Ausnahme, die ausgelöst wird, wenn ein Programme ungültige MSIL\-Befehle \(Microsoft Intermediate Language\) oder Metadaten enthält. Normalerweise deutet dies auf einen Fehler im Compiler hin, von dem das Programm generiert wurde.
