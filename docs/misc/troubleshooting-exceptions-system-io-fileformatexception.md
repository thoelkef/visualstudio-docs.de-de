# Problembehandlung bei Ausnahmen: System.IO.FileFormatException
Eine <xref:System.IO.FileFormatException>\-Ausnahme, die ausgelöst wird, wenn eine Eingabedatei oder ein Datenstream nicht mit einer bestimmten Dateiformatspezifikation übereinstimmt.
