# Problembehandlung bei Ausnahmen: System.MulticastNotSupportedException
Diese Ausnahme wird bei dem Versuch ausgelöst, zwei Instanzen eines nicht kombinierbaren Delegattyps zu kombinieren. Es wird keine Ausnahme ausgelöst, wenn einer der Operanden ein NULL\-Verweis ist \(`Nothing` in Visual Basic\).
