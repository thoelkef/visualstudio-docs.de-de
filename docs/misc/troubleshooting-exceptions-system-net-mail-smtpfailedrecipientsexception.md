# Problembehandlung bei Ausnahmen: System.Net.Mail.SmtpFailedRecipientsException
Diese Ausnahme wird ausgelöst, wenn eine E\-Mail mit einem <xref:System.Net.Mail.SmtpClient> versendet wird und nicht an alle Empfänger ausgeliefert werden kann.
