# Problembehandlung bei Ausnahmen: System.Net.NetworkInformation.NetworkInformationException
Die Ausnahme, die ausgelöst wird, wenn während des Abrufens von Netzwerkinformationen ein Fehler auftritt.  
  
## Hinweise  
 Typen im <xref:System.Net.NetworkInformation>\-Namespace lösen diese Ausnahme aus, wenn ein Aufruf einer Win32\-Funktion fehlschlägt. Die <xref:System.Net.NetworkInformation.NetworkInformationException.ErrorCode%2A>\-Eigenschaft enthält das von der Funktion zurückgegebene Ergebnis.
