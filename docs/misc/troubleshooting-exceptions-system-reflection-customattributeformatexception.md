# Problembehandlung bei Ausnahmen: System.Reflection.CustomAttributeFormatException
Die Ausnahme, die ausgelöst wird, wenn das Binärformat eines benutzerdefinierten Attributs ungültig ist.  
  
## Hinweise  
 Ursache kann ein Fehler in einem Tool sein, das zur Erstellung des Datentyps verwendet wurde, der den Fehler auslöst.
