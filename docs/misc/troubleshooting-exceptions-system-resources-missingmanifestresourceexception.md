# Problembehandlung bei Ausnahmen: System.Resources.MissingManifestResourceException
Diese Ausnahme wird ausgelöst, wenn die Hauptassembly nicht die Ressourcen für die neutrale Kultur enthält, diese jedoch benötigt werden, da die entsprechende Satellitenassembly fehlt.
