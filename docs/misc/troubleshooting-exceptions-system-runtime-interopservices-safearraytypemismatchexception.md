# Problembehandlung bei Ausnahmen: System.Runtime.InteropServices.SafeArrayTypeMismatchException
Diese Ausnahme wird ausgelöst, wenn der Typ eines eingehenden `SAFEARRAY` nicht mit dem Typ übereinstimmt, der in der verwalteten Signatur festgelegt ist.
