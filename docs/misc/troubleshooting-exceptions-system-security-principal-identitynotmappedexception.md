# Problembehandlung bei Ausnahmen: System.Security.Principal.IdentityNotMappedException
Diese Ausnahme wird ausgelöst, wenn ein Principal keine zugeordnete Identität besitzt und keiner bekannten Identität zugeordnet werden kann.
