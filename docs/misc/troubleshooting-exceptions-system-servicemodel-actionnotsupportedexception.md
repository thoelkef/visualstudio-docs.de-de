# Problembehandlung bei Ausnahmen: System.ServiceModel.ActionNotSupportedException
Eine <xref:System.ServiceModel.ActionNotSupportedException>\-Ausnahme wird in der Regel auf dem Client ausgelöst, wenn die mit dem aufgerufenen Vorgang zusammenhängende Aktion nicht mit den Aktionsvorgängen auf dem Server übereinstimmt.
