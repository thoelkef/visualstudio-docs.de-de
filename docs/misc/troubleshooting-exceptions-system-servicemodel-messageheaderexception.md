# Problembehandlung bei Ausnahmen: System.ServiceModel.MessageHeaderException
Eine <xref:System.ServiceModel.MessageHeaderException>\-Ausnahme wird ausgelöst, wenn die Erwartungen bezüglich der Header einer SOAP\-Nachricht bei der Verarbeitung der Nachricht nicht erfüllt werden.
