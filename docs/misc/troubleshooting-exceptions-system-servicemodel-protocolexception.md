# Problembehandlung bei Ausnahmen: System.ServiceModel.ProtocolException
Eine <xref:System.ServiceModel.ProtocolException>\-Ausnahme tritt am Client auf, wenn die Kommunikation mit der Gegenseite aufgrund von nicht übereinstimmenden Datenübertragungsprotokollen nicht möglich ist.  
  
## Hinweise  
 Die <xref:System.ServiceModel.ProtocolException> kann durch nicht kompatible Bindungen zwischen Endpunkten, die Verwendung einer falschen Adresse des Remoteendpunkts oder eine in ähnlicher Weise falsche Konfiguration verursacht werden.
