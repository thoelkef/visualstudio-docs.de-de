# Problembehandlung bei Ausnahmen: System.ServiceModel.Security.SecurityNegotiationException
Eine <xref:System.ServiceModel.Security.SecurityNegotiationException>\-Ausnahme gibt an, dass ein Fehler aufgetreten ist, während der Sicherheitskontext für eine Nachricht ausgehandelt wurde.
