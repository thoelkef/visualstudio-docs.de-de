# Problembehandlung bei Ausnahmen: System.Transactions.TransactionAbortedException
Diese Ausnahme wird ausgelöst, wenn versucht wird, eine Operation in einer Transaktion durchzuführen, für die bereits ein Rollback ausgeführt wurde.
