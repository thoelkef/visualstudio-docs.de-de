# Problembehandlung bei Ausnahmen: System.Web.Services.Protocols.SoapHeaderException
Diese Ausnahme wird ausgelöst, wenn eine XML\-Webdienstmethode über SOAP aufgerufen wird und eine Ausnahme während der Verarbeitung des SOAP\-Headers auftritt.
