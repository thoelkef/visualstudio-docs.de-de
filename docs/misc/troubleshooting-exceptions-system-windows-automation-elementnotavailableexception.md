# Problembehandlung bei Ausnahmen: System.Windows.Automation.ElementNotAvailableException
Eine <xref:System.Windows.Automation.ElementNotAvailableException>\-Ausnahme enthält Informationen zu der Ausnahme, die ausgelöst wird, wenn versucht wird, auf ein UI\-<xref:System.Windows.Automation.AutomationElement> zuzugreifen, das einem nicht mehr verfügbaren Teil der Benutzeroberfläche entspricht.  
  
## Hinweise  
 Diese Ausnahme kann ausgelöst werden, wenn sich das Element in einem geschlossenen Dialogfeld oder in einer geschlossenen oder beendeten Anwendung befindet.
