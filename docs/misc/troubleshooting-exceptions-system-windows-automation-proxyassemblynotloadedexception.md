# Problembehandlung bei Ausnahmen: System.Windows.Automation.ProxyAssemblyNotLoadedException
Eine <xref:System.Windows.Automation.ProxyAssemblyNotLoadedException>\-Ausnahme enthält Informationen zu einer Ausnahme, die ausgelöst wird, wenn beim Laden einer Assembly mit clientseitigen Anbietern ein Problem auftritt.
