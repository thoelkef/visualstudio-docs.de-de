# Problembehandlung bei Ausnahmen: System.Windows.Controls.PrintDialogException
Die <xref:System.Windows.Controls.PrintDialogException>\-Ausnahme wird ausgelöst, wenn beim Öffnen oder beim Verwenden eines <xref:System.Windows.Controls.PrintDialog> bzw. beim Zugriff auf diesen ein Fehlerzustand auftritt.
