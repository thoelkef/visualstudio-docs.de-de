# Problembehandlung bei Ausnahmen: System.Workflow.Activities.Rules.RuleSetValidationException
Eine <xref:System.Workflow.Activities.Rules.RuleSetValidationException>\-Ausnahme wird ausgelöst, wenn die Fortsetzung einer Verarbeitung nicht möglich ist, da ein Regelsatz nicht überprüft werden kann. Die Ausnahme wird vom <xref:System.Workflow.Activities.Rules.RuleEngine>\-Konstruktor ausgelöst, wenn der verwendete <xref:System.Workflow.Activities.Rules.RuleSet> Validierungsfehler aufweist.
