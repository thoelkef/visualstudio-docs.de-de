# Problembehandlung bei Ausnahmen: System.Workflow.ComponentModel.Serialization.WorkflowMarkupSerializationException
Eine <xref:System.Workflow.ComponentModel.Serialization.WorkflowMarkupSerializationException>\-Ausnahme wird von Methoden ausgelöst, die die Serialisierung und die Deserialisierung ausführen. Die Ausnahme schließt Informationen über den Zeilen\- und Spaltenspeicherort ein, der bewirkt hat, dass die Ausnahme ausgelöst wird.
