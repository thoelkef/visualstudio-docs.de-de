# Unable to delete file &lt;name&gt;. The file may be in use by another application or marked read-only.
Im Allgemeinen tritt dieser Fehler auf, wenn das zum Löschen ausgewählte Tastaturschema verwendet wird oder schreibgeschützt ist.  
  
### So beheben Sie diesen Fehler  
  
1.  Wenn die angegebene Datei durch eine andere Anwendung verwendet wird, schließen Sie diese.  
  
     \- oder \-  
  
     Wenn die Datei schreibgeschützt ist, entfernen Sie den Schreibschutz.  Sie können den Schreibschutz im Eigenschaftendialogfeld für die Datei ändern, verfügbar im Datei\-Explorer.
