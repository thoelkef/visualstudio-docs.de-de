# Grundlegendes zu VSPackages
VSPackages sind Softwaremodule, die Visual Studio um Benutzeroberflächenelemente, Dienste, Projekte, Editoren und Designer erweitern. VSPackages sind die zentrale architektonische Einheit von [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)].[!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] selbst wird hauptsächlich in Form einer Auflistung von VSPackages geschrieben.  
  
 Zum Erstellen einer [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)]\-Erweiterungsanwendung erstellen Sie zuerst ein VSPackage. Dann können Sie Menübefehle, Toolfenster Dokumentfenster oder anderen Code hinzufügen, um die [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)]\-IDE zu erweitern.  
  
 Um VSPackages verwenden zu können, müssen Sie das Visual Studio SDK installieren. Um diese exemplarische Vorgehensweise befolgen zu können, müssen Sie das Visual Studio SDK installieren. Weitere Informationen finden Sie unter [Visual Studio SDK](../extensibility/visual-studio-sdk.md).
