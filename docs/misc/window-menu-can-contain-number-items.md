# Window menu can contain &lt;number&gt; items.
Die Fensterliste im Fenstermenü kann nur bis zu 24 Fensternamen enthalten.  Im Allgemeinen tritt dieser Fehler auf, wenn im Dialogfeld Optionen unter Umgebung im Bereich Allgemein im Feld **Fenstermenü enthält \<Anzahl\> Elemente** eine Zahl kleiner als 1 oder größer als 24 eingegeben wurde.  
  
### So beheben Sie diesen Fehler  
  
1.  Geben Sie im Feld eine Zahl zwischen 1 und 24 ein.
