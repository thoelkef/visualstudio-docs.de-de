# Wizard file &lt;name&gt; is corrupted.
Die **VSZ**\-Datei für den Assistenten hat nicht das richtige Format.  Dieser Fehler tritt auf, wenn eine VSZ\-Datei beschädigt ist oder wenn die Datei von einem anderen Produkt als Visual Studio stammt.  
  
### So beheben Sie diesen Fehler  
  
1.  Stellen Sie sicher, dass der Assistent ein Visual Studio\-Assistent ist.  
  
     – oder –  
  
     Installieren Sie Visual Studio erneut, um die beschädigte Datei zu ersetzen.
