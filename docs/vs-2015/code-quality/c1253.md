# <a name="c1253"></a>C1253
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warnung C1253: Modelldatei konnte nicht geladen.  
  
 Das Codeanalysetool meldet diese Warnung aus, wenn ein interner Fehler in der Modelldatei, nicht im analysierten Code vorhanden ist.
