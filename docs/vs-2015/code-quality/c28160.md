# <a name="c28160"></a>C28160
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Die neueste Version dieses Themas finden Sie unter [C28160](https://docs.microsoft.com/visualstudio/code-quality/c28160).  
  
Warnung C28160: Fehleranmerkung  
  
 Diese Warnung wird ausgegeben, wenn eine `__drv_error` Anmerkung wurde gefunden. Diese Anmerkung wird verwendet, um die Praktiken zur codeerstellung zu kennzeichnen, die korrigiert werden soll, und kann verwendet werden, mit einem `__drv_when` Anmerkung, um bestimmte Kombinationen von Parametern anzugeben.
