# <a name="c28204"></a>C28204
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warnung C28204: \<Funktion >: nur eine dieser Überschreibung und unter \<Filename > (\<Zeile >) sind mit Anmerkungen für \<Paramname >: beide oder keines von beiden versehen werden müssen.  
  
 Diese Warnung wird ausgegeben, wenn ein Fehler, in den Anmerkungen vorliegt.
