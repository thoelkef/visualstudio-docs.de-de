# <a name="c28208"></a>C28208
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warnung C28208: Funktion \<Funktion > wurde zuvor mit einer anderen Parameterliste bei definiert \<Datei > (\<Zeile >). Manche Analysetools können falsche Ergebnisse liefern.  
  
 Diese Warnung wird ausgegeben, wenn eine bekannte Funktionsdefinition nicht mit einem weiteren Vorkommen übereinstimmt.
