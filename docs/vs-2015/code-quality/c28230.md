# <a name="c28230"></a>C28230
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warnung C28230: Der Parametertyp weist keinen Member auf.  
  
 Diese Warnung gibt an, dass ein Argument in eine Anmerkung versucht, den Zugriff auf eine `struct`, `class`, oder `union` und das benannte Element ist nicht vorhanden.
