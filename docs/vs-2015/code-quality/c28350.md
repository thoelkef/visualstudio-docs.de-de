# <a name="c28350"></a>C28350
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warning C28350: die Anmerkung \<annotation> beschreibt eine Situation, die nicht bedingt anwendbar ist.  
  
 Diese Warnung wird normalerweise generiert, wenn eine Anmerkung angewendet wird, bei der der C/C++-Typ überprüft wird.
