# <a name="ca0505"></a>CA0505

Die veraltete Eigenschaft wird ignoriert, da die superceding-Eigenschaft definiert ist.

Warnung CA505 wird generiert, wenn die angegebenen Eigenschaften für veraltete und übergeordnete Code Analysen Werte in einer Projekt-oder buildkonfigurationsdatei enthalten. Um dieses Problem zu beheben, entfernen Sie die veraltete-Eigenschaft.
