# <a name="idiaenumframedatareset"></a>IDiaEnumFrameData::Reset
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Die neueste Version dieses Themas finden Sie unter [idiaenumframedata:: Reset](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiaenumframedata-reset).  
  
Setzt eine Enumerationsfolge auf den Anfang zurück.  
  
## <a name="syntax"></a>Syntax  
  
```  
HRESULT Reset();  
```  
  
## <a name="return-value"></a>Rückgabewert  
 Gibt `S_OK`zurück.
