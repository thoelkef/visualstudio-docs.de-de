Die neueste Version dieses Themas finden Sie unter [idiasymbol:: Get_symtag](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiasymbol-get-symtag).

## <a name="syntax"></a>Syntax  
  
```cpp#

   DWORD* pRetVal
```  
