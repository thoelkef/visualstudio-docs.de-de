# <a name="flowchart-activity-designers"></a>Flussdiagramm-Aktivitätsdesigner
[!INCLUDE[wfd1](../includes/wfd1-md.md)] umfasst eine Reihe von vom System bereitgestellten Aktivitäten, die Sie zum Erstellen von Workflows verwenden können. Dieser Abschnitt enthält die vom System bereitgestellten Aktivitäten, die zum Verwalten von Flussdiagrammen innerhalb eines Workflows dienen. In den folgenden Themen werden diese Aktivitäten beschrieben und Anleitungen zu ihrer Verwendung bereitgestellt.  
  
## <a name="in-this-section"></a>In diesem Abschnitt  
 [Flussdiagramm](../workflow-designer/flowchart-activity-designer.md)  
 **Verwendet, um Workflows zu erstellen, die definieren und verwalten komplexe flusssteuerungen.**  [FlowDecision](../workflow-designer/flowdecision-activity-designer.md)  
 Stellt eine Verzweigung für den Steuerungsverlauf in eine von zwei Alternativen bereit, deren Auswahl darauf basiert, ob eine angegebene Bedingung erfüllt wird.  
  
 [FlowSwitch\<T>](../workflow-designer/flowswitch-t-activity-designer.md)  
 Stellt einen Branch für den Steuerungsverlauf bereit, der auf der Übereinstimmung mit Kriterien basiert, wenn mehr als zwei alternative Branches erforderlich sind.
