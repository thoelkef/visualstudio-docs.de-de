# <a name="transaction-activity-designers"></a>Transaktionsaktivitätsdesigner

Transaktionsaktivitätsdesigner werden verwendet, um Transaktionsaktivitäten, z. B. das Kompensieren oder Bestätigen, zu erstellen und zu konfigurieren. Die Themen, die in diesem Abschnitt enthaltenen und in der folgenden Tabelle aufgeführten enthalten Anleitungen zur Verwendung von Workflow-Designer-Aktivität und den Vorlagen-Designer.
