# <a name="invalid-number"></a>Ungültige Zahl
Ungültige Anzahl (ErrbadNumber 1013)  
  
 Sie haben versucht, eine Zahl zu verwenden, die in JavaScript nicht gültig ist.  
  
 Informationen über gültige Nummern finden Sie unter [Number-Objekt](../../javascript/reference/number-object-javascript.md).
