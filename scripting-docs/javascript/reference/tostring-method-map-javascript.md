# <a name="tostring-method-map-javascript"></a>toString-Methode (Map) (JavaScript)
Gibt eine Zeichenfolgendarstellung einer Zuordnung zurück.  
  
## <a name="syntax"></a>Syntax  
  
```JavaScript  
mapObj.toString()  
```  
  
#### <a name="parameters"></a>Parameter  
 `mapObj`  
 Erforderlich. Ein `Map`-Objekt.  
  
## <a name="property-valuereturn-value"></a>Eigenschaftswert/Rückgabewert  
 Die Zeichenfolgendarstellung der Zuordnung.
